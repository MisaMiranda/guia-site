Passo a passo completo

Do zero ao site no ar

Aprenda a criar, publicar e atualizar seu site usando Claude, Git, GitHub e Vercel — do zero, sem complicação.

Passo 1

Preparar o arquivo

Nomeie o arquivo como index.html desde o início

Crie uma pasta para o projeto (ex: meu-site)

Mova o index.html para dentro dessa pasta

O nome index.html é o padrão que todo servidor web reconhece automaticamente.
Passo 2

Instalar o Git no seu PC

Acesse git-scm.com e baixe o instalador para Windows

Clique em Next em todas as telas (pode deixar tudo no padrão)

Para verificar se instalou, abra o terminal e digite:

git --version
Se aparecer 'git version 2.x.x', está instalado e funcionando.
Passo 3

Criar conta e repositório no GitHub

Acesse github.com e crie uma conta gratuita

Clique em + (canto superior direito) > New repository

Dê um nome ao repositório (ex: meu-site), marque Public

Clique em Create repository

O nome de usuário vai aparecer no endereço dos seus projetos — escolha algo simples.
Passo 4

Abrir o terminal na pasta do projeto

Abra o terminal: Windows + R > digite cmd > Enter

Navegue até a pasta do projeto:

cd "Desktop\meu-site"

Ou se estiver em Downloads:

cd "Downloads\meu-site"
Passo 5

Iniciar o Git e enviar para o GitHub

git init
git add index.html
git commit -m "primeiro commit"

Cole a linha que o GitHub mostra na página do repositório:

git remote add origin https://github.com/seu-usuario/meu-site.git
git branch -M main
git push -u origin main
Atenção: tem um espaço entre 'commit' e '-m'. Copie a linha do remote add direto do GitHub.

Passo 6

Publicar no Vercel

Acesse vercel.com e clique em Sign Up > Continue with GitHub

Na tela inicial clique em Add New > Project

Encontre seu repositório na lista e clique em Import

Deixe todas as opções no padrão e clique em Deploy

Pronto! Seu site estará no ar em segundos

Seu link ficará no formato: https://meu-site.vercel.app — compartilhe com qualquer pessoa!

Git fica instalado no seu computador. Claude, GitHub e Vercel ficam na nuvem. O Git é a única peça que vive na sua máquina — e é ele que faz a ponte entre o que você criou localmente e o que o mundo vê online.
Com prática, esse fluxo vira segunda natureza.
